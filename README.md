# 8-Queens Puzzle Solver

An interactive browser-based puzzle where you place 8 queens on a chessboard so no two queens attack each other.

## Features
- Place and remove queens by clicking on the board
- Real-time conflict detection
- Hint system to guide your next move
- Browse all 92 solutions

## How to Play
1. Click any cell to place a queen
2. Click again to remove it
3. Place all 8 queens without any conflicts
4. Use Hint button if stuck

## Tech Stack
HTML | CSS | JavaScript
